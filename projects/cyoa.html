CYOA Project

Context

In this project I had to make a story where you get to choose what to do. I had to make multiple files in sub directories that had links that would bring you to other files to make it work.


Content

I made a story where the goal was to prank Mr. Mueller. You would make multiple choices which would lead you to fail at your goal or to different ways of you pranking Mr. Mueller. Using HTML I made links from one file to the next. Here is an exmple of the links used in the first file, door.html:


                <a> href = "computer/at-computer.html">Walk to your seat</a>

                <a> href = "exit-classroom/hall-pass.html">Ask for the hall pass</a>

Challenges

A challenge I had was when I was trying to make links from my endings folder back to my door.html file. The problem was that the link i made didn't work because I forgot to make it go back up to a parent directory first. Here was my origional link:

<a> href = "door.html">Restart</a>

To fix it I had to add ../, which would tell the link to first go out of the endings folder so it would then be in the paren directory of door.html. Here is the fixed link:

<a> href = "../door.html">Restart</a>


Takeaways

A takeaway from this challenge is that you need to be very direct in the path you make your links take to get to their destination like with my link to the door.html file. I first need to make the link go it it's parent directory of the file that I want to go to.


Next Steps

My next step for this project would be to add more of a background or images. This is because as the state that I have left this project at it is just an MVP (Minimum Viable Product) and it is just text with links and a white background.
