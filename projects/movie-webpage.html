Movie Webpage Project

In this project I had to make a webpage for a movie trailer. We had to make up a storie for the movie and then code a website around it using HTML and CSS.

With HTML we used tags such as headings, paragraphs, divisions, spans, inputs, and more.

With CSS we changed and added things such as the colors of background and text, boarders, margins, padding, and fonts.

A challenge while making this projct was using background because we wanted to make the background fade from one color to the next, so we had to use `linear-gradient`.
