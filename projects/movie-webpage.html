Movie Webpage Project

Context

In this project I had to make a webpage for a movie trailer. We had to make up a story for the movie and then code a website around it using HTML and CSS.

Content

We made a movie trailer for an action movie called The Clash of the World. The story for the movie was that the main character, The Son of Liberty, wanted to seek revenge on the person who killed his parents and he discovers his powers allong the way. Using HTML, we made headings, paragraphs, divisions, spans, inputs, and more. With CSS, we changed and added things such as the colors of background and text, boarders, margins, padding, and fonts.

Challenges

A challenge I faced while making this projct was using background because we wanted to make the background fade from one color to the next, so we had to use linear-gradient. On top of that we had to make the color of the text contrast with the color of the background so the text was easy to read.

Takeaways

A takeaway from this challenge is that you should be careful of waht colors you choose while making a webpage because having poor contrast between your text and background can make it hard to read.

Next Steps

My next steps for this project would be to add a second webpage as we did not have enough time to make it. In the second webpage would be a sequel to the first movie with a new story that we would use the same CSS code to make.
