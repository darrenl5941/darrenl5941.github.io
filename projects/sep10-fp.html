SEP 10 Freedom Project

Context

In this project we got to choose a topic of our choice. We would have to find the current technology, both software and hardware, in that topic and come up with possible future technology in our topic as well. We would then have to make a website to show and talk about the current and future technology we came up with in our topic.


Content

For this project I made a website about the Stock Market. My website would show multiple peices of software and hardware in the stock market such as different stock market apps, monitors, quantum devices, and feild-programmable gate arrays. In my website I used everything I learned from my software engineering course (HTML, CSS, and Bootstrap) to make my website. I also used a tool in coding that I learned on my own, jQuery, to make my website more unique from others doing the project. I used jQuery to make parts of my website alternate what information it would show by clicking on it. Here is an example of the code I used to do this:


                $(".monitor-2").hide();
                $(".monitor-3").hide();

                $("body").on("click", "#monitors", function(){

                    if ($(".monitor-1").is(":visible")) {

                        $(".monitor-1").hide();
                        $(".monitor-2").fadeIn();

                    } else if ($(".monitor-2").is(":visible")) {

                        $(".monitor-2").hide();
                        $(".monitor-3").fadeIn();

                    } else if ($(".monitor-3").is(":visible")) {

                        $(".monitor-3").hide();
                        $(".monitor-1").fadeIn();

                    }

                });
